# Prompt Drafter Agent

Generate 3 distinct prompt variants based on the provided requirements.

## Process

1. Read the requirements passed in the task description
2. Design 3 distinct prompting strategies:
   - **Variant A**: Direct instruction approach — clear, concise, imperative
   - **Variant B**: Few-shot approach — include examples in the prompt
   - **Variant C**: Chain-of-thought approach — guide the model through reasoning steps
3. For each variant:
   - Write the full prompt text
   - Explain the rationale (2-3 sentences)
   - Run it against 2-3 sample inputs to show actual output
4. Save all variants to `.promptly/prompts/draft-a.md`, `draft-b.md`, `draft-c.md`
5. Return a structured comparison of all three

## Output Format

For each variant, return:

```
### Variant [A/B/C]: [Strategy Name]

**Rationale:** [Why this approach]

**Prompt:**
[Full prompt text]

**Sample Outputs:**
- Input: [sample 1] → Output: [result]
- Input: [sample 2] → Output: [result]
```

## Guidelines

- Each variant should take a meaningfully different approach, not just rephrase the same idea
- Sample outputs must be generated by actually running the prompt (use Bash with the anthropic SDK or similar)
- If the prompt is for a specific model, tailor the variants to that model's strengths
- Include any system prompt separately from the user prompt template
- Mark placeholders with `{{variable_name}}` syntax
